# spotlightWallpaper
![image](https://raw.githubusercontent.com/shuli495/resource/master/spotlightWallpaper/spotlightWallpaper.png)

**支持条件：**
1. win10系统
2. 开启聚焦壁纸

**使用方法：**
1. 开启win10锁屏聚焦壁纸，开始菜单 - 设置 - 个性化 - 锁屏界面 - 背景选择“Windows 聚焦”
2. 打开本软件，点击“壁纸存放路径”，软件会自动同步一次壁纸
3. 点击图片可直接切换壁纸
4. 可最小化到托盘

**软件介绍：**\
c# WPF 编写(.NET4.0 )\
